# api-rest
API para estudos de REST

Aula 01 - Como iniciar um projeto novo! 
- Utilizando o npm init -y, para criação do arquivo package.json
- Utilizando o npm install e o nome do módulo que irá utilizar, para poder adicionar estes módulos na pasta node_modules e poder utilizar.
- Criação de arquivo app.js.
- Realização da criação de conexão com servidor.
- Criação de rota através do app.get e de leitura através do app.listen.
- Instalação do módulo nodemon, que torna possível a verificação em tempo real das modificações que são realizadas no projeto!

Aula 02 - Cirar Rotas GET com Express, Refatorando Código, Extensão JSON Viewer
- Como criar rotas de requisição GET para exibir textos simples, JSON e status code.
- Criação de arquivo server para subir servidor.
- Utilização de Mock para mocar dados para testes.

Aula 03 - Criação de Rota POST e leitura de JSON com REST Client 
- Como utilizar rota POST.
- Realização de leitura de arquivos JSON.
- Teste de API com REST Client
